## Description

- **Platform**: iOS
- **Minimum iOS Version**: iOS 16

## Installation
1. Install dependencies at terminal
`pod install`
2. Open `Mureng.xcworkspace` in XCode
3. Build and run in XCode


## Features
- 오늘의 영어 질문에 대한 답 작성
- 오늘의 표현 학습
- 영어 질문 작성하기
- 다른 사람들의 답변 확인하기
- 잉크 수집하기
